::selection

Select text to preview the selection background color

Select any part of this page to preview how the current accent color changes the text selection highlight. This demo makes it easy to compare each theme option and see which color feels the most readable.

A good selection color should stand out clearly without overpowering the text itself. Lighter tones often feel softer, while stronger tones can make interaction states more obvious and energetic.

Try dragging across a single sentence, then across multiple paragraphs, to test how the highlight behaves with longer selections. Small differences in contrast become much easier to notice during extended reading.

This kind of detail is subtle, but it improves the polish of a reading interface. Thoughtful selection styling helps turn a basic page into something that feels intentionally designed and pleasant to use.
